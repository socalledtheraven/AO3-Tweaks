AO3 Tweaks Options

Rekudos

Adds the option to automatically comment with a variety of supportive messages after you have left kudos the first time.

  • Automatically comments when you click the kudos button after having already left kudos.

Comments

Adds a button that shows various templates for things to comment about, and prewritten gushing comments, if you aren't sure what to say.

Adds additional comment boxes inbetween chapters, so you don't forget what you wanted to talk about when viewing a fic in fulltext.

Latest Chapter

Adds a button on your bookmarks page, when sorted by Date Updated, which takes you to the most recent chapter published, so you can just read what's new.

To Read

Adds a button, both to a work's page and the page of a search or filter, that automatically saves that work as a bookmark with the tag "To Read", which can then be looked through with the "Bookmarker's Tags" option at your leisure. Enjoy having much fewer tabs open!

  • Automatically unsubscribes from a saved work, for if you're transferring over from saving works with subscriptions.

  • Removes the native mark for later buttons, which are superfluous with this addon.

  • Adds a "Save privately" button, which creates a private bookmark instead, for the works you'd prefer others be unaware you're considering.

  • Adds a "Mark as Read" button, which removes the "To Read" tag from the work and leaves it as an ordinary bookmark.

Bookmark Notes

Adds the details of each work to the bookmark's notes. Useful for if a work is deleted after the fact.

  • Carries over your existing notes when updating bookmarks.

  • Add the title of the work and the username (and pseud if applicable) of the creator, with relevant links to each, to the bookmark notes. Highly recommended, as bookmark notes are not deleted when a work is deleted, and finding a lost work is easier with the title, username, and url.

  • Adds the summary of the work to the notes.

  • Autocheck the Rec checkbox to make the bookmark a rec.

  • Autocheck the Private checkbox to make the bookmark private.

  • Add a copy of the relationship categories to the bookmarker tag list.

  • Add a copy of the fandom tags to the bookmarker tag list.

  • Add a copy of the character tags to the bookmarker tag list.

  • Add a copy of the relationship tags to the bookmarker tag list.

  • Add a copy of the additional/freeform tags to the bookmarker tag list.

  • Add a copy of the rating (with "Rating: " before it) to the bookmarker tag list.

  • Add a copy of the archive warnings to the bookmarker tag list.

  • Add a tag with the exact word count.

Adds the details of each series to the bookmark's notes. Useful for if a series is deleted after the fact.

  • Add the title of the series and the username(s) (and pseud(s) if applicable) of the creator(s) with relevant links to each to the bookmark notes.

  • Add the number of works in the series to the tags.

  • Add a tag with the exact word count.

  • Autocheck the Rec checkbox to make the bookmark a rec on series bookmarks.

  • Autocheck the Private checkbox to make the bookmark private on series bookmarks.
